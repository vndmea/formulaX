# Third-Party Notices

## Kity Vendor Packages

This document acknowledges third-party components used in FormulaX.

### Kity Formula Source

The FormulaX project imports source code from the following kity projects:

- **kf-editor**: Kity formula editor core
- **kf-parser**: Kity formula LaTeX parser
- **kf-render**: Kity formula rendering engine

These packages are imported into `packages/kity-vendor-*` directories.

## External Dependencies

FormulaX depends on the following open-source packages:

- **KaTeX**: LaTeX math rendering (Apache License 2.0)
- **Tiptap**: Rich text editor framework (Apache License 2.0)
- **TinyMCE**: Web-based rich text editor (LGPL 2.1)
- **Vitest**: Unit testing framework (MIT License)
- **Playwright**: Browser testing framework (Apache License 2.0)
