# Kity Vendor Editor

Vendor package for kf-editor source.

## Source

This package contains imported source from the kf-editor project.

## Files

- `vendor/src` - Original source code
- `vendor/assets` - Static assets
- `vendor/build` - Build configuration
- `vendor/resource` - Resource files
- `vendor/lib` - Library dependencies
- `vendor/config` - Configuration files
- `vendor/dev-lib` - Development utilities
